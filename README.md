# Сделанное

## prepared.py

Состоит из 2 функций:
### To16x9
Скрипт, подгоняющий все изображения и видеофайлы в указанной папке под разрешение 1920х1080, путём добавления на задний план размытой затемнёной копии
### Convert
Конвертирует:
jgp, jpeg, webp -> png
svg -> png
gif -> mp4
avi -> mp4
